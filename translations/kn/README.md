## 🌱 ಪ್ರಾರಂಭಿಸುವುದು

ಈ ಕೋರ್ಸ್ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ರೂಪಿಸುವ ಮೂಲಭೂತಗಳನ್ನು ಒಳಗೊಂಡ ಪಾಠಗಳನ್ನು ಹೊಂದಿದೆ. ಪ್ರತಿ ಪಾಠವು ಅದರದೇ ವಿಷಯವನ್ನು ಹೊಂದಿದೆ ಆದ್ದರಿಂದ ನಿಮ್ಮ ಆಶ್ರಯ ಇರುವಲ್ಲಿ ಪ್ರಾರಂಭಿಸಿ!

ಈ ಕೋರ್ಸ್‌ಗೆ ಬಹುಭಾಷಾ ಬೆಂಬಲವಿದೆ. ನಮ್ಮ [ಲಭ್ಯವಿರುವ ಭಾಷೆಗಳು ಇಲ್ಲಿ](#-multi-language-support) ನೋಡಿ.

ನೀವು ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳೊಂದಿಗೆ ಮೊದಲ ಬಾರಿ ಕೆಲಸ ಮಾಡುತ್ತಿರಾದರೆ, ನಮ್ಮ [ಜನರೇಟಿವ್ AI ಆರಂಭಿಕರಿಗಾಗಿ](https://aka.ms/genai-beginners) ಕೋರ್ಸ್ ನೋಡಿ, ಇದರಲ್ಲಿ ಜನ AI ಉಪಯೋಗಿಸುವ 21 ಪಾಠಗಳಿವೆ.

ಈ ರೆಪೊಗೆ [ಸ್ಟಾರ್ (🌟) ಕೊಡುವುದನ್ನು ಮರೆಯಬೇಡಿ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ಮತ್ತು [ಈ ರೆಪೊವನ್ನು ಫೋರ್ಕ್ ಮಾಡಿ](https://github.com/microsoft/ai-agents-for-beginners/fork) ಕೋಡ್ ಅನ್ನು ಚಲಾಯಿಸಲು.

### ಇತರೆ ಕಲಿಕಾರ್ತೆಗಳನ್ನು ಭೇಟಿ ಮಾಡಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳ ಉತ್ತರಗಳನ್ನು ಪಡೆಯಿರಿ

ನೀವು ಅಲುಗಾಡಿದರೆ ಅಥವಾ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ರೂಪಿಸುವ ಬಗ್ಗೆ ಯಾವುದೇ ಪ್ರಶ್ನೆಗಳಿದ್ದರೆ, ನಮ್ಮ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ನಲ್ಲಿ ನಿಷ್ಠುರವಾದ Discord ಚಾನೆಲ್ ಸೇರಿರಿ.

### ನೀವು ಬೇಕಾಗಿರುವುದು

ಈ ಕೋರ್ಸ್‌ನ ಪ್ರತಿಯೊಂದು ಪಾಠದಲ್ಲೂ ಕೋಡ್ ಉದಾಹರಣೆಗಳು ಇದ್ದವೆ, ಅದು code_samples ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ದೊರೆಯುತ್ತದೆ. ನೀವು [ಈ ರೆಪೊನ್ನು ಫೋರ್ಕ್](https://github.com/microsoft/ai-agents-for-beginners/fork) ಮಾಡಿಕೊಂಡು ನಿಮ್ಮ ಪ್ರತಿಯನ್ನು ರಚಿಸಬಹುದು.

ಈ ವ್ಯಾಯಾಮಗಳಲ್ಲಿ ಬಳಕೆಯಾದ ಕೋಡ್ ಉದಾಹರಣೆಗಳು Microsoft Agent Framework ಅನ್ನು Azure AI Foundry Agent Service V2 ಜೊತೆಗೆ ಬಳಸುತ್ತವೆ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) – Azure ಖಾತೆ ಬೇಕು

ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ Microsoft ನಿಂದ ಈ ಕೆಳಗಿನ AI ಏಜೆಂಟ್ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು ಮತ್ತು ಸೇವೆಗಳನ್ನು ಬಳಸಲಾಗುತ್ತಿದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ಈ ಕೋರ್ಸ್‌ಗಾಗಿ ಕೋಡ್ ಎಗ्जಿಕ್ಯೂಟ್ ಮಾಡುವ ಕುರಿತು ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗೆ, [Course Setup](./00-course-setup/README.md) ನೋಡಿ.
