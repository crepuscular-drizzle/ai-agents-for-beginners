> **స్థానికంగా క్లోన్ చేయాలనుకుంటున్నారా?**
>
> ఈ రిపోజిటరీ 50+ భాషల అనువాదాలను కలిగి ఉంటుంది, ఇది డౌన్లోడ్ పరిమాణాన్ని గణనీయంగా పెంచుతుంది. అనువాదాలు లేకుండా క్లోన్ చేయడానికి, స్పార్స్ చెకౌట్ ఉపయోగించండి:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> ఇది కోర్సును పూర్తి చేయడానికి అవసరమైన ప్రతిదాన్నీ చాలా వేగంగా డౌన్లోడ్ చేయడానికి అవకాశం ఇస్తుంది.

## 🌱 ప్రారంభించడం

ఈ కోర్సులో AI ఏజెంట్లు నిర్మించడాన్ని గురించి మౌలిక పాఠాలు ఉన్నాయి. ప్రతి పాఠం దాని స్వంత విషయం మీద కవర్ చేస్తుంది కాబట్టి మీరు ఇష్టమయిన చోట నుండి ప్రారంభించండి!

ఈ కోర్సుకు బహుభాషా మద్దతు ఉంది. మా [అందుబాటులో ఉన్న భాషలను ఇక్కడ చూడండి](#-multi-language-support).

మీరు మొదటిసారీ జనరేటివ్ AI మోడల్స్ తో నిర్మిస్తున్నట్లయితే, మా [జనరేటివ్ AI ప్రారంభికుల కోసం](https://aka.ms/genai-beginners) కోర్సును చూడండి, ఇందులో GenAI తో 21 పాఠాలు ఉన్నాయి.

ఈ రెపోను [స్టార్ చేయడం (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) మరియు [ఫోర్క్ చేయడం](https://github.com/microsoft/ai-agents-for-beginners/fork) మర్చిపోకండి, తద్వారా కోడ్‌ను నడుపుతారు.

### ఇతర నేర్చుకునేవారిని కలవండి, మీ ప్రశ్నలకు జవాబులు పొందండి

మీరు అడ్డుకొంతా లేదా AI ఏజెంట్లను నిర్మించడంలో ఏవైనా ప్రశ్నలు ఉన్నట్లయితే, మా ప్రత్యేకమైన Discord చానెల్ Microsoft Foundry Discord లో చేరండి ([లింక్](https://aka.ms/ai-agents/discord)).

### మీకు కావలసినవి

ఈ కోర్సు లో ప్రతి పాఠంలో కోడ్ ఉదాహరణలు ఉంటాయి, అవి code_samples ఫోల్డర్ లో పొందుపరచబడ్డాయి. మీరు [ఈ రెపోను ఫోర్క్](https://github.com/microsoft/ai-agents-for-beginners/fork) చేసుకుని మీ సొంత కాపీ సృష్టించుకోవచ్చు.

ఈ వ్యాయామాల్లో కోడ్ ఉదాహరణలు Microsoft Agent Framework ను Azure AI Foundry Agent Service V2 తో ఉపయోగిస్తుంది:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ака운్ట్ అవసరం

ఈ కోర్సు Microsoft నుండి క్రింది AI ఏజెంట్ ఫ్రేమ్‌వర్క్‌లు మరియు సర్వీసులను ఉపయోగిస్తుంది:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ఈ కోర్సు కోసం కోడ్ నడపడం గురించి మరింత సమాచారం కోసం, [Course Setup](./00-course-setup/README.md) చూడండి.

## సహకారం

ఈ ప్రాజెక్ట్ సహకారాలను మరియు సూచనలను ఆహ్వానిస్తుంది. ఎక్కువ భాగం సహకారాలు మీరు క్రియేటర్ లైసెన్స్ ఒప్పందం (CLA) కు అంగీకరిస్తారని, మీరు మీ సహకారాన్ని ఉపయోగించడానికి హక్కులు కలిగివున్నారని, మరియు మీ సహకారంలో మా ఉపయోగించడానికి హక్కులను వాస్తవానికి ఇస్తున్నారని ప్రకటించే ఒక ఒప్పందంపై అంగీకరించాలని అవసరం. వివరాలకు, సందర్శించండి <https://cla.opensource.microsoft.com>.

మీరు పుల్ రిక్వెస్ట్ సమర్పించినప్పుడు, CLA బాట్ ఒక నిర్దేశిస్తుంది మీరు CLA అందించవలసిన అవసరముంటే, మరియు PR కోసం తగిన అలంకరణ (ఉదా., స్థితి తనిఖీ, వ్యాఖ్య) ను సృష్టిస్తుంది. బాట్ అందించే సూచనలను అనుసరించండి. ఈ పని మీరు మా CLAని ఉపయోగించే అన్ని రిపోజిటరీలలో సారిగా ఒక్కసారి చేయాలి.
